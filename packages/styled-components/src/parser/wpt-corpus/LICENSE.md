# Web Platform Tests — vendored fixtures

`corpus.json` in this directory is a derivative work mechanically
extracted from CSS parsing tests in the Web Platform Tests (WPT)
repository at the pinned commit below. It is used solely as fixture
data for parity testing against browser CSS semantics; nothing under
this directory ships in the published package.

## Source

- Upstream: https://github.com/web-platform-tests/wpt
- Pinned commit: `e18ee881f51ec9beca2702345c0869a0aa69cdc5`
